# Revenue Optimization Project

This project analyzes restaurant operational data to identify key drivers of revenue and evaluate how staffing, promotions, and customer behavior impact performance.

## Key Insights

- Staffing levels between 8–10 servers produce the most consistent performance during peak demand  
- Customer volume is the primary driver of revenue  
- Promotions increase traffic but reduce average ticket size
